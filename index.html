Intro

So, you're here to learn about the game Celeste for whatever reason. If that's not right (or I didn't show this to you) you should probably go. Hello remaining few!

Celeste is a platform game in which players control a girl named Madeline as she makes her way up a mountain while avoiding various deadly obstacles and overcoming her inner demons. You'll find abandoned citys, temples with dark magic, caves of cyrstal bliss and a hotel run by a lost soul.

As of 2020, Celeste has sold over 1,000,000 copies. The original version of the game was made over 4 days as a game jam. You can find this original game hidden as an easter egg in the most recent version today. The came has won over 11 incredible awards and been nominated for 30 others. The entire game was designed for speedrunning. Featureing hidden passages, harder (but shorter) levels and great rewards for precision accuracy. The game also gives you all the hidden moves in the beginning. You don't unlock them. Once you learn them, you can use them anywhere.

For more info about the gameplay and plot go the the info page.
